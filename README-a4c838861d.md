# 🎓 高等数学教学互动系统

> 基于GitHub Actions的自动化教学内容生成与管理系统

---

## 📌 系统简介

本系统通过GitHub Actions实现教学内容的自动化生成，涵盖学生反馈分析、数学试卷生成、课堂白板创建和考研复习计划生成四大核心功能，支持高等数学12个完整章节，覆盖基础教学到考研冲刺的全场景需求。

### ✨ 核心特性

- **零服务器成本**：基于GitHub Actions，无需自建服务器
- **完全自动化**：定时触发+手动触发双模式
- **数据驱动**：基于学生反馈和学习数据优化内容
- **考研专用**：针对数学一/二/三精准定位重点章节
- **响应式设计**：支持PC/移动端，LaTeX公式即时渲染
- **进度跟踪**：本地存储学习进度，支持导出报告

## 🚀 快速开始

### 1. 前置要求
- GitHub账号
- 基本的Git操作能力

### 2. 部署步骤（5分钟）
```bash
# 1. 克隆或Fork本仓库
git clone [你的仓库地址]

# 2. 将文件推送到GitHub
git push origin main

# 3. 在GitHub仓库中启用Actions
# 进入Settings > Actions > General > Workflow permissions
# 选择"Read and write permissions"
```

### 3. 触发工作流
进入仓库的 **Actions** 页面，选择对应工作流，点击 **Run workflow**

📖 详细部署指南：[QUICK_START.md](QUICK_START.md)

## 🎯 功能模块

### 1. 学生反馈分析
- 自动统计带"反馈"标签的Issues
- 按课程和章节统计反馈数量
- 分析学生参与度和平均评价
- 生成每日/每周趋势统计

**触发方式**：每天8点自动运行，或手动触发

**输出结果**：`stats.json` 数据文件 + HTML可视化页面

### 2. 数学试卷生成
- 内置微积分、线性代数、概率统计题库
- 支持从GitHub Issues读取自定义题库
- 按难度等级筛选题目
- 随机生成试卷并自动打乱顺序

**触发方式**：手动触发

**输出结果**：试卷HTML页面

### 3. 课堂白板生成
- 内置12个章节完整模板
- 包含学习目标、核心概念、重要公式、典型例题、课堂练习
- 支持LaTeX公式渲染
- 支持响应式布局和PDF导出

**触发方式**：手动触发或Release发布时自动触发

**输出结果**：白板HTML页面

### 4. 考研复习内容生成
- 针对数学一/二/三精准定位
- 支持基础、强化、冲刺三个复习阶段
- 基于历年真题分析重点章节
- 集成学习进度跟踪功能

**触发方式**：手动触发

**输出结果**：复习计划HTML页面 + 进度跟踪数据
